# 🖥️ PC Setup

Format sonrası tek tıkla program kurulum aracı.

## Programlar
- Revo Uninstaller
- Everything
- WizTree
- WinRAR
- Google Chrome

## Kullanım

### Hazır EXE (Releases'dan indir)
1. Sağ tarafta **Releases** → **PC-Setup.exe** indir
2. Çift tıkla, çalıştır

### Kendin build et
1. Python yükle: [python.org](https://python.org/downloads)
2. `build.bat` çift tıkla
3. `dist/PC-Setup.exe` hazır
